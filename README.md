# WPF的空项目模板

该项目包含了下面的一些常用的工具包，有助于快速的开发项目。

### 包含功能包列表：

- Community ToolKit Mvvm :一个Mvvm的工具类可以帮你快速的开发Mvvm项目，并完成数据都通知和事件的绑定功能。
- SqlSugar :一个数据库的工具类，方便你对数据库的所有操作。
- Handy Control ：一个挺好看的界面控件库，实现了好多好看的控件。
- Host:对程序生命周期的管理，和注册各种服务。

### 使用方法：
1. 将项目Clone到本地用Visual Studio打开。
2. 选择菜单上的项目->导出模板->点击下一步->给模板起个名字，确认，自动将模板导入到Visual Studio是勾选状态->点击完成。
3.重新打开Visual Studio->新建项目，搜索你刚刚给模板起的名字，就是使用模板了。
